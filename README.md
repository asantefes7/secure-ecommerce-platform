# Secure E-Commerce Platform with Fraud Detection

## Overview
This capstone project involves developing a full-stack web application that simulates an online e-commerce store, emphasizing robust security measures and intelligent fraud detection. The platform will allow users to browse products, manage shopping carts, and complete secure checkouts, while incorporating machine learning to identify and prevent fraudulent transactions. This project aligns with applied computer science by integrating web development, database management, machine learning, and cybersecurity practices.

## Project Objectives
- Design and implement a secure e-commerce app adhering to industry standards.
- Integrate ML for real-time fraud detection.
- Demonstrate full-stack skills.
- Evaluate performance and produce documentation.

(Full details in proposal document.)

## Tech Stack
- Frontend: React.js
- Backend: Node.js with Express.js
- Database: MongoDB
- Payment: Stripe API (test mode)
- Security: JWT, bcrypt, helmet.js, OWASP ZAP
- Fraud Detection: Python with scikit-learn/TensorFlow
- Other: Git, Docker, Postman, Heroku/AWS
